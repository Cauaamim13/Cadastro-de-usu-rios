📋 Sistema de Cadastro de Usuários

Aplicação web para cadastro e gerenciamento de usuários, desenvolvida com React + Vite.
🚀 Tecnologias Utilizadas

React - Biblioteca JavaScript para construção de interfaces
Vite - Build tool moderna e rápida
Axios - Cliente HTTP para requisições à API
CSS3 - Estilização da interface

✨ Funcionalidades

✅ Cadastrar novos usuários (nome, idade e email)
✅ Listar todos os usuários cadastrados
✅ Excluir usuários
✅ Interface responsiva e intuitiva
✅ Integração com API REST

🎯 Características Técnicas

React Hooks: useState, useEffect, useRef
Comunicação com API: GET, POST e DELETE
Renderização dinâmica: Lista de usuários atualizada em tempo real
Componentização: Código organizado e reutilizável
